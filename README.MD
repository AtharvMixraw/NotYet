# NotYet

A visual job rejection tracker that turns the frustration of "not yet" responses into a colorful, interactive board.
![alt text](<img.png>)
## Overview

NotYet is a single-page web application designed to help job seekers track and visualize their rejections during the application process. Instead of letting rejection emails pile up in your inbox, NotYet transforms them into colorful pins on a digital corkboard, creating a visual representation of your job search journey.

## Features

- Pin-based rejection tracking with company name, role, and rejection date
- Drag-and-drop interface for organizing pins on the canvas
- Customizable pin colors (yellow, pink, blue, green, orange)
- Dark mode support
- Scrollable canvas for unlimited pins
- Export board as high-resolution PNG image
- Persistent storage using browser localStorage
- Responsive layout with fixed sidebar and scrollable canvas

## Technology Stack

- HTML5
- CSS3 with Tailwind CSS
- Vanilla JavaScript
- html2canvas for image export
- Google Fonts (Inter)
- Material Symbols for icons

## Installation

No installation required. Simply open the HTML file in any modern web browser.

For local development:
1. Clone or download the repository
2. Open `index.html` in your browser
3. Start tracking your rejections

## Usage

1. Fill in the company name, role, and rejection date in the left panel
2. Select a pin color from the available options
3. Click "Create Pin" to add it to the board
4. Drag pins around the canvas to organize them as desired
5. Click "Export Board as Image" to download your rejection board
6. Use "Clear All Pins" to reset the board

## Design Philosophy

NotYet reframes job rejection as a visual marker of persistence and growth. Every pin represents effort, learning, and progress toward the right opportunity. The colorful, playful design helps normalize rejection as part of the job search process while creating a shareable artifact of your journey.

## Browser Compatibility

Works on all modern browsers supporting ES6 JavaScript, HTML5, and CSS3.
